[Theme-UI](https://theme-ui.com/components) provides a wide set of primitive
components to build reusable components. This naturally makes it
easy to work with a React-based framework like Next.js. In this tutorial,
I'll show you how to make use of the `next/link` and `next/image` components in
any Theme-UI project.
